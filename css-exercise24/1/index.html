The display property is used to specify how an element is shown on a web page.

inline:An inline element DOES NOT start on a new line and only takes up as much width as necessary.Such as:span,a,img

block: A block-level element ALWAYS starts on a new line and takes up the full width available (stretches out to the left and right as far as it can).Such as: div,p,h1 to h6,form,section,footer and...

none: The element is completely removed
