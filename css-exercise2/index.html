Selector:CSS selectors are used to "find" (or select) the HTML elements you want to style.
declaration:declaration is the key-value pair of a CSS property and its value. CSS declarations are used to set style properties and construct rules to apply to individual or groups of elements.
property: A CSS property determines an HTML element's style or behavior.
value: The value defines that style, and is specific to the property, like red or 10 pixels
